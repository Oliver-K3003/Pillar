# Setup Instructions

---

## Dependencies

### Prereqs

Install [NodeJS LTS](https://nodejs.org/en)

Install [Python 3.13](https://www.python.org/downloads/)

### Python

Run the following command

```bash
pip install -r /path/to/requirements.txt
```

### Node

Run the following command

```bash
npm i
```

## Running the App

Enter the following into the terminal

```bash
npm run dev
```
